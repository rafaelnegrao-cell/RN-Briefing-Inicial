# Starter de Briefing — RN Negrão

Este é o modelo-base para criar um briefing novo. Veja o passo a passo completo
no arquivo **GUIA-DO-PADRAO-RN.md** (na pasta acima).

Resumo do que trocar por cliente:
1. `public/assets/emblema.png`  → logo do cliente
2. `public/config.js`           → nome, cor da marca, textos
3. `public/questions.js`        → perguntas adaptadas ao segmento
Depois: subir no GitHub + Railway (PostgreSQL é opcional).
